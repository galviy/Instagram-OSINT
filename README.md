# Setting up cookies
you can modify the `token.json` to your information details, SHARING THIS DETAILS COULD LEAD UR ACCOUNT GETTING STOLEN!
```json
{
  "sessionid": "",
  "csrftoken": "",
  "ds_user_id": ""
}
```
